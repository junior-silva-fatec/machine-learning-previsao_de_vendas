Resumo

O estudo analisa métodos de previsão de vendas no varejo, comparando algoritmos de aprendizado supervisionado, como Regressão Linear Múltipla, Random Forest e SVM. Usando dados de 45 lojas da Walmart, considera variáveis como histórico de vendas, feriados e preço de combustível, visando ajudar empresários a melhorar estratégias de marketing e promoção.


Metodologia

O estudo analisa métodos de previsão de vendas no varejo, comparando algoritmos de aprendizado supervisionado, como Regressão Linear Múltipla, Random Forest e SVM. Usando dados de 45 lojas da Walmart, considera variáveis como histórico de vendas, feriados e preço de combustível, visando ajudar empresários a melhorar estratégias de marketing e promoção.


A. Conjunto de Dados

Os dados de vendas considerados para o modelo de previsão foram foram retirados de 45 lojas da Walmart. Os dados históricos de previsão abrangem as vendas de 5 de fevereiro de 2010 a 1 de novembro de 2012 [25]. Existem 3 ficheiros de dados separados correspondentes a cada ano e a exatidão dos modelos foi foi calculada em conformidade. O conjunto de dados considerado para o estudo contém os seguintes campos:


  1. Loja - o número de lojas, uma vez que são consideradas 45 lojas consideradas.
  2. Data - Considerámos a data como a primeira data da semana de vendas para a previsão de séries cronológicas.
  3. Vendas semanais - vendas semanais para uma determinada loja.
  4. Sinal de feriado - para determinar se a semana é uma semana de feriado. Mostra 1 para semana de feriados e 0 para semana sem feriados. Isto ajudará a compreender as tendências durante os feriados.
  5. Temperatura - Temperatura registada no dia da venda.
  6. Preço do combustível - Custo do combustível na região onde a loja está localizada.
  7. IPC - Índice de preços ao consumidor dominante.
  8. Desemprego - Taxa de desemprego dominante na região onde a loja está presente.

B. Passos para o experimento

  1. O primeiro passo é importar as bibliotecas necessárias que necessárias que serão utilizadas para construir o modelo, tais como numpy, pandas, matplotlib, seaborn.
  2. Depois disso, carregámos o conjunto de dados para cada ano de 2010, 2011 e 2012, respetivamente, para o IDE.
  3. Depois de os dados terem sido carregados, preparou-os para a experiência, convertendo a data para o formato datetime.
  4. Verificar se existem valores em falta ou nulos.
  5. De seguida, dividimos a coluna da data e criámos 3 colunas nomeadamente dia, mês e ano.
  6. Para construir o modelo de previsão, encontrámos os valores anómalos nos dados, traçando a temperatura, o preço do combustível, o IPC e o Desemprego no eixo X.
  7. O passo seguinte foi eliminar os valores anómalos e considerar o intervalo em que os valores anómalos não se enquadram.
  8. Em seguida, verificou-se novamente se o gráfico está correto sem os valores anómalos.
  9. A biblioteca sklearn foi importada para construir o modelo e selecionou caraterísticas e objectivos para os eixos X e Y para prever as vendas.
  10. Dividiu os dados em conjuntos de treino e de teste na proporção de 80:20.
  11. Utilizou a regressão linear, o regressor Random Forest, KNN Regressor, SVR, Extra Tree Regressor para prever as vendas da loja Walmart no eixo Y para efetuar uma análise comparativa análise comparativa do modelo de previsão.
  12. Calculou os erros no modelo de previsão calculando o erro absoluto médio, o erro quadrático médio e a raiz do erro quadrático médio.
